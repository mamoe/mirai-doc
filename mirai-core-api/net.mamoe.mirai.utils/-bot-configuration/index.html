Bot 配置.

Kotlin 使用方法:

val bot = BotFactory.newBot(...) {
   // 在这里配置 Bot

   bogLoggerSupplier = { bot -> ... }
   fileBasedDeviceInfo()
   inheritCoroutineContext() // 使用 `coroutineScope` 的 Job 作为父 Job
}

Java 使用方法:

Bot bot = BotFactory.newBot(..., new BotConfiguration() {{
    setBogLoggerSupplier((Bot bot) -> { ... })
    fileBasedDeviceInfo()
    ...
}})
