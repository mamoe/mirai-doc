GlobalEventChannel

object GlobalEventChannel : EventChannel<Event>
Content copied to clipboard

全局事件通道. 此通道包含来自所有 Bot 的所有类型的事件. 可通过 EventChannel.filter 过滤得到范围更小的 EventChannel.
