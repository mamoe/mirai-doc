interface Friend : User, CoroutineScope
Content copied to clipboard

代表一位好友.

一个 Friend 实例并不是独立的, 它属于一个 Bot. 对于同一个 Bot, 任何一个人的 Friend 实例都是单一的. Friend 无法通过任何方式直接构造. 任何时候都应从 Bot.getFriend 或事件中获取.
