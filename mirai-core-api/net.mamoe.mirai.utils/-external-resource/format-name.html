formatName

abstract val formatName: String
Content copied to clipboard

文件格式，如 "png", "amr". 当无法自动识别格式时为 DEFAULT_FORMAT_NAME.

默认会从文件头识别, 支持的文件类型: png, jpg, gif, tif, bmp, amr, silk
