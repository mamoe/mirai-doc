queryUrl

suspend fun Image.queryUrl(): String
Content copied to clipboard

查询原图下载链接.

  • 当图片为从服务器接收的消息中的图片时, 可以直接获取下载链接, 本函数不会挂起协程.

  • 其他情况下协程可能会挂起并向服务器查询下载链接, 或不挂起并拼接一个链接.

Return

原图 HTTP 下载链接

Throws

kotlin.IllegalStateException

当无任何 Bot 在线时抛出 (因为无法获取相关协议)
