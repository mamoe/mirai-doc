Image

interface Image : Message, MessageContent, CodableMessage
Content copied to clipboard

自定义表情 (收藏的表情) 和普通图片.

最推荐的存储方式是存储图片原文件, 每次发送图片时都使用文件上传. 在上传时服务器会根据其缓存情况回复已有的图片 ID 或要求客户端上传. 详见 Contact.uploadImage

根据 ID 构造图片

  • Image.fromId. 在 Kotlin, 更推荐使用顶层函数 val image = Image("id")

上传和发送图片

  • Contact.uploadImage 上传 资源文件 并得到 Image 消息

  • Contact.sendImage 上传 资源文件 并发送返回的 Image 作为一条消息

  • ExternalResource.uploadAsImage

  • ExternalResource.sendAsImageTo

  • Contact.sendImage

下载图片

  • Image.queryUrl 扩展函数. 查询图片下载链接

  • IMirai.queryImageUrl 查询图片下载链接 (Java 使用)

mirai 码支持

格式: &#91;mirai:image:Image.imageId&#93;
