newStranger

abstract fun newStranger(bot: Bot, strangerInfo: StrangerInfo): Stranger
Content copied to clipboard

构造一个 Stranger 对象. 它持有对 Bot 的弱引用(WeakRef).

Bot 无法管理这个对象, 但这个对象会以 Bot 的 Job 作为父 Job. 因此, 当 Bot 被关闭后, 这个对象也会被关闭.
