移动端

在移动客户端将会显示为卡片<title>: DisplayStrategy.generateTitle<preview>: DisplayStrategy.generatePreview<summary>: DisplayStrategy.generateSummary
|-------------------------|
| <title>                 |
| <preview>               |
|-------------------------|
| <summary>               |
|-------------------------|
默认显示方案:
|-------------------------|
| 群聊的聊天记录             |
| <消息 1>                 |
| <消息 2>                 |
| <消息 3>                 |
|-------------------------|
| 查看 3 条转发消息          |
|-------------------------|

PC 端

在部分 PC 端显示为类似移动端的卡片, 在其他 PC 端显示为以下格式
鸽子 A 2020/04/23 11:27:54

咕
