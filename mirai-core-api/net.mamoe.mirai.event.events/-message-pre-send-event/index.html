MessagePreSendEvent

sealed class MessagePreSendEvent : AbstractEvent, BotEvent, BotActiveEvent, CancellableEvent

在发送消息前广播的事件. 可被 取消.

此事件总是在 MessagePostSendEvent 之前广播.

当 MessagePreSendEvent 被 取消 后:

  • MessagePostSendEvent 不会广播

  • 消息不会发送.

  • Contact.sendMessage 会抛出异常 EventCancelledException
